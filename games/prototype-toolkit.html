Overview

Prototype Toolkit is a set of editor tools (level editor, dialogue sequencer and balance dashboard) used to accelerate iteration and prototyping.

Features

  • Level editor with drag & drop
  • Dialogue sequencer with branching
  • Live tuning dashboard

Integration

Integrates with Unity and Godot pipelines; includes sample projects and documentation.
