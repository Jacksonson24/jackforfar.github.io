First-Class Dissertation Project

Unreal Engine 5 | Solo Final-Year Project | First-Class Grade

Project Overview

Developed as my final-year dissertation, this solo Unreal Engine 5 project is a cooperative survival-horror experience set on a hostile desert planet. Players work together to explore an abandoned facility, restore critical systems and survive against a dangerous creature.

The project focused on environmental storytelling, immersive interaction and cooperative gameplay. Facility layouts were generated procedurally, while power, oxygen and communications objectives encouraged players to explore, communicate and divide responsibilities.

My Responsibilities

  • Designed and developed the project independently in Unreal Engine 5
  • Created gameplay systems using Blueprint Visual Scripting
  • Implemented cooperative multiplayer functionality
  • Developed the procedural facility-generation system
  • Created power, oxygen and satellite communication objectives
  • Designed diegetic watch and terminal interfaces
  • Built environments and environmental storytelling elements
  • Conducted structured playtesting and iterated based on feedback

Key Features

  • Cooperative Multiplayer
  • Procedural Generation
  • Survival Horror
  • Diegetic Interfaces
  • Environmental Storytelling
  • Objective Systems
  • Blueprint Development
  • Structured Playtesting

Outcome

The project was awarded a First-Class dissertation grade. Development was documented through more than 100 progress videos, supporting testing, bug tracking and continued feature iteration throughout production.
